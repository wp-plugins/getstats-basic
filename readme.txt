Know your visitors better: Post content that is relevant to the geo location of your audience based on the getStats stats plugin! 

== Description ==

Find the right times to post based on your readers location, give your readers more targeted content and add value to your blog! Perfect for optimizing ads on your blog. Know where your traffic comes from and use this data to deliver better content and ads.


= Benefits & Key Features =

* Minimum Page Load
* Very user friendly & Easy to use
* Don't hurt your page load speed unlike other analytics software
* Tracks visitor country, ip, referrer, user agent, visiting time


== Installation ==

Using the WordPress dashboard

1. Login to admin panel
2. Go to Plugins
3. Select Add New
4. Search Simple visitor stat
5. Select Install Now
6. Select Activate Plugin

Manual

1. Download the plugin and extract the files
2. Upload the directory "getstats-basic" to your wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Check the "Visitor stat" Tab created by this plugin for view visitor stat

That's all. You dont need to do anything else. Enjoy!
